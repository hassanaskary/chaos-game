# chaos-game
This web app generates and visualizes the Chaos Game. It can generate various
fractals including:
- Sierpinski Triangle
- Sierpinski Hexagon
- Sierpinski Carpet
- Pentagon EH 1 (from wikipedia)
- Barnsley Fern
